# E-Commerce Sales Performance Analysis

## Project Overview

This project analyzes e-commerce sales data using Python.

## Technologies Used

- Python
- Pandas
- Matplotlib
- JupyterLab

## Dataset

The dataset contains:

- Order ID
- Customer
- Region
- Category
- Quantity
- Unit Price
- Discount

## Analysis Performed

- Data cleaning
- Feature engineering
- KPI calculation
- Sales analysis
- Profit analysis
- Regional performance
- Customer analysis
- Data visualization
- Business insights

## Key Features

- Handled missing values
- Calculated Gross Sales, Net Sales, and Profit
- Compared regional and category performance
- Identified top customers
- Exported processed data for reporting
